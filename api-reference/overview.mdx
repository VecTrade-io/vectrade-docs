# API Reference

The VecTrade API is organized around REST. All requests use `https://api.vectrade.io/v1` as the base URL and return JSON responses.

## Base URL

```
https://api.vectrade.io/v1
```

## Authentication

All endpoints require an API key via the `X-API-Key` header:

```bash
X-API-Key: vq_your_key_here
```

Get your key at [vectrade.io/vtrade/developer](https://vectrade.io/vtrade/developer). See the [Authentication Guide](/guides/authentication) for key management best practices.

## Quick Summary

| Group | Endpoints | Description |
|-------|-----------|-------------|
| **Quotes** | `GET /vq/quotes/{symbol}`, `POST /vq/quotes/batch` | Real-time and batch price data |
| **Fundamentals** | `GET /vq/fundamentals/{symbol}`, `/statements` | Financial statements and ratios |
| **Profile** | `GET /vq/profile/{symbol}` | Company info, sector, industry, exchange |
| **Technicals** | `GET /vq/technical/{symbol}` | 50+ indicators with configurable intervals |
| **News** | `GET /vq/news/{symbol}` | Market news with sentiment scoring |
| **Sentiment** | `GET /vq/sentiment/{symbol}` | Social & news sentiment signals |
| **Analyst** | `GET /vq/analyst-consensus/{symbol}`, `/analyst-targets/{symbol}`, `/upgrades-downgrades/{symbol}` | Wall Street consensus data |
| **Earnings** | `GET /vq/earnings/{symbol}` | Earnings history and surprise records |
| **Insider** | `GET /vq/insider/{symbol}` | Insider trading activity |
| **Options** | `GET /vq/options/{symbol}/chain` | Options chains with Greeks |
| **Historical** | `GET /vq/quotes/{symbol}/history` | OHLCV price history |
| **ETF** | `GET /vq/etf/{symbol}` | ETF profile, NAV, and holdings info |

See [Rate Limits Guide](/guides/rate-limits) for optimization strategies.

## Errors

All errors return a consistent JSON structure:

```json
{
  "error": {
    "type": "rate_limit_error",
    "message": "Rate limit exceeded",
    "request_id": "req_abc123",
    "retry_after": 2
  }
}
```

See [Error Handling Guide](/guides/error-handling) for the full error taxonomy and retry strategies.
