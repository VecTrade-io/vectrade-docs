# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in VecTrade documentation (e.g.,
exposed secrets in code examples), please report it responsibly:

1. **Do NOT** open a public issue
2. Email **security@vectrade.io** with details
3. Include the page URL and description

We will acknowledge receipt within 48 hours.

## Scope

This repository contains documentation source. For vulnerabilities in the
platform or SDKs, report to the respective repositories.
